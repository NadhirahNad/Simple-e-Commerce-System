# Simple E-Commerce System

A simple Python Tkinter-based GUI application that simulates a basic e-commerce system. Users can enter their personal details, view available products, add items to a shopping cart, clear the cart, and view the cart summary with total price.

## Features
- User information input (Name, Phone, Address)
- Product listing
- Add selected products to cart
- Delete all items from cart
- View cart contents and total price
- Popup messages for user actions

## Technologies Used
- Python
- Tkinter (GUI)

## How to Run
1. Make sure Python is installed.
2. Tkinter comes pre-installed with Python.
3. Run the application:
   python ecommerce_app.py

## Notes
- Currency used is RM (Ringgit Malaysia).
- This project is for learning and demonstration purposes.
